## Idea
The Problem :
Most AI productivity assistants generate emails, meeting replies, task summaries, and schedules without learning from user feedback. Users repeatedly make the same corrections to writing style, tone, formatting, and preferences because every request is treated as a completely new interaction. Existing AI assistants also consume unnecessary tokens by repeatedly processing entire documents or project files, leading to increased API costs, slower performance, and inefficient context management. Additionally, when an AI model reaches token limits or becomes unavailable, many assistants simply fail instead of continuing the user's work.

The Solution : Persona — A Personal Productivity Agent That Learns From Your Corrections
Persona AI is a lightweight CLI-based adaptive productivity assistant that continuously improves through a transparent feedback loop. Whenever a user edits or rejects AI-generated content, Persona AI analyzes the correction, extracts the underlying writing preference, and stores it as a reusable rule instead of memorizing the exact edit. These learned preferences are automatically applied to future requests, and the assistant explicitly informs the user which preference is being used before generating the response.

To improve efficiency, Persona AI integrates Graphify, which converts project folders, notes, documents, PDFs, and codebases into a structured knowledge graph. Instead of processing entire files for every request, Persona AI retrieves only the most relevant contextual information required for the current task, significantly reducing token usage while improving response quality.

Persona AI also intelligently selects the most suitable AI model based on the user's task. If the selected model reaches token limits, API quotas, or becomes unavailable, the system automatically switches to the next-best available model without interrupting the workflow, ensuring a reliable and uninterrupted user experience.
